## [1.0.0] - 2025-01-22

### Initial Production Release

The Analyst - AI-powered analytics orchestration platform for media companies.

#### Features
- **Multi-agent Architecture** - 9 specialized agents with single responsibilities
- **Human-in-Loop** - Advisory mode ensures transparency and control
- **Statistical Rigor** - All analyses include methodology, confidence intervals, limitations
- **Arabic NLP** - MARBERT + CAMeL Tools for Arabic text analysis
- **Predictive Modeling** - Time series forecasting, classification, regression
- **BORIS Compliance** - 94/100 score on multi-agent governance framework

#### Components
- Main orchestrator coordinating 8 specialized sub-agents
- CLI interface with 5 slash commands
- PostgreSQL database integration
- Desktop and Obsidian notifications
- Comprehensive test suite (831+ tests, 84.50% coverage)
